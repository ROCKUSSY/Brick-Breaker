# CG Mini-Project Brick Breaker

### You Need These Files First :
* sudo apt-get install freeglut3-dev
* sudo apt-get install libsdl1.2-dev libsdl-image1.2-dev

## To Run The File:

#### Open Terminal in Ubuntu

$ g++ filename.cpp -lGL -lGLU -lglut
$ ./a.out
